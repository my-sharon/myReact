这是一个前后端分离的基于react脚手架开发的后台管理系统

运行这个后台管理系统需要以下条件
1：配置好MongoDB环境，开启MongoDB服务
2：安装MongoDB的可视化工具MongoChef(studio-3t-x64)用来存放数据库数据，
打开可视化工具创建一个连接，连接到MongoDB服务器localhost：27017，
创建数据库lk_data,把data文件夹中的json文件引入lk_data
3：运行manager_api，在此文件夹打开终端,npm run dev 运行服务器
4：运行react脚手架my-manager，在此文件夹打开终端,npm start运行项目
